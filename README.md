# baiduAdfinisher
去除百度搜索内容页广告chrome扩展程序(兼容各大“国产”浏览器急速版).

 <div>###离线chrome扩展程序（.crx文件）安装方式： </div>
 <div>1、打开chrome扩展程序页面 chrome://extensions/ 或者 菜单 > 更多工具> 扩展程序；</div>
 <div>2、把。crx文件拖进该页面（国产浏览器直接拖入任意页面）。</div>
